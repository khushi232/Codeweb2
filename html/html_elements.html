HTML Elements

An HTML element is defined by a start tag, some content, and an end tag.

HTML Element

An HTML element is defined by a start tag, some content, and an end tag:


<tagname> Content goes here </tagname>


The HTML element is everything from the start tag to the end tag:

<h4> My First heading </h4>
<p> My First paragraph. </p>

Start Tag Element content End Tag
<h4> My First Heading </h4>
<p> My First paragraph. </p>
<br> none none

Note: Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!

Nested HTML Elements

HTML elements can be nested (this means that elements can contain other elements).

All HTML documents consist of nested HTML elements.

The following example contains four HTML elements (<html>, <body>, <h4> and <p>):

Example

OUTPUT

My First Heading

My First paragraph.

HTML

<!DOCTYPE html>
<html>
<body>

<h4> My First heading </h4>
<p> My First paragraph. </p>

</body>
</html>

Example Explained

The <html>element is the root element and it defines the whole HTML document.

It has a start tag <html> and an end tag </html>.

Then, inside the <html> element there is a<body> element:

Example

OUTPUT

My First Heading

My First paragraph.

HTML

<body>

<h4> My First heading </h4>
<p> My First paragraph. </p>

</body>

The <body> element defines the document's body.

It has a start tag <body> and an end tag </body>.

Then, inside the <body> element there are two other elements: <h4> and <p>:

Example

OUTPUT

My First Heading

My First paragraph.

HTML

<h4> My First heading </h4>
<p> My First paragraph. </p>

The <h4> element defines a heading.

It has a start tag <h4> and an end tag </h4> :

Example

OUTPUT

My First heading

HTML

<h4> My First heading </h4>

The <p> element defines a heading.

It has a start tag <p> and an end tag </p> :

Example

OUTPUT

My First paragraph

HTML

<p> My First paragraph </p>

Never Skip the End Tag

Some HTML elements will display correctly, even if you forget the end tag:

Example

OUTPUT

This is a paragraph.

This is a paragraph.

HTML

<html>
<body>

<p> This is a paragraph
<p> This is a paragraph

</body>
</html>

However, never rely on this! Unexpected results and errors may occur if you forget the end tag!

Empty HTML Elements

HTML elements with no content are called empty elements.

The <br> defines a line break, and is an empty element without a closing tag:

Example

OUTPUT

This is a
paragraph with a line break.

HTML

<p> This is a <br> paragraph with a line break. </p>

HTML is Not Case Sensitive

HTML tags are not case sensitive: <P> means the same as <p> .

The HTML standard does not require lowercase tags, but W3C recommends lowercase in HTML, and demands lowercase for stricter document types like XHTML.
