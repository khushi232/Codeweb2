HTML Styles

< Previous Next >

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

The HTML Style Attribute

Setting the style of an HTML element, can be done with the style attribute.

The HTML style attribute has the following syntax:

Example

HTML

<tagname style ="property value;">

The property is a CSS property. The value is a CSS value.

Example

OUTPUT

I am normal

I am red

I am blue

I am big

HTML

<p> I am normal </>p>
<p style ="color : red;"> I am red </>p>
<p style ="color : blue;"> I am blue </>p>
<p style ="font-size : 50px;"> I am big </>p>

Background Color

The CSS background-color property defines the background color for an HTML element.

Example

Set the background color for a page to powderblue:

OUTPUT

This is a heading

This is a paragraph.

HTML

<!DOCTYPE html>
<html>
<body style ="backgroundcolor : powderblue;">

<h4> This is a heading </h4>
<p> This is a paragraph. </p>

</body>
</html>

Example

Set background color for two different elements:

OUTPUT

This is a heading

This is a paragraph.

HTML

<h4 style ="backgroundcolor : powderblue;"> This is a heading </h4>
<p style ="backgroundcolor : tomato;"> This is a paragraph. </p>

Text Color

The CSS color property defines the text color for an HTML element:

Example

OUTPUT

This is a heading

This is a paragraph.

HTML

<h4 style ="color : blue;"> This is a heading </h4>
<p style ="color : red;"> This is a paragraph. </p>

Fonts

The CSS font-family property defines the font to be used for an HTML element:

Example

OUTPUT

This is a heading

This is a paragraph.

HTML

<h4 style ="font-family : verdana;"> This is a heading </h4>
<p style ="font-family : courier;"> This is a paragraph. </p>

Text Size

The CSS font-size property defines the text size for an HTML element:

Example

OUTPUT

This is a heading

This is a paragraph.

HTML

<h4 style ="font-size : 300%;"> This is a heading </h4>
<p style ="font-size : 160%;"> This is a paragraph. </p>

The CSS text-align property defines the horizontal text alignment for an HTML element:

Example

OUTPUT

Centered Heading

Centered paragraph.

HTML

<h4 style ="text-align : center;"> This is a heading </h4>
<p style ="text-align : center;"> This is a paragraph. </p>
